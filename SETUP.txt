RESTAURANT WEBSITE - QUICK START GUIDE

BACKEND SETUP
=============
1. cd backend
2. npm install
3. Create .env file with:
   PORT=5000
   MONGODB_URI=mongodb://localhost:27017/restaurant
   JWT_SECRET=your_secret_key
   EMAIL_USER=your-email@gmail.com
   EMAIL_PASSWORD=your-app-password
4. npm run dev (runs on http://localhost:5000)

FRONTEND SETUP
==============
1. cd frontend (new terminal)
2. npm install
3. npm run dev (runs on http://localhost:3000)

FEATURES
========
- Online Ordering with Cart
- Table Reservations
- User Auth (Register/Login)
- Order Tracking
- Email Notifications
- Admin Dashboard
- Responsive Design

DATABASE SETUP
==============
Local: mongodb://localhost:27017/restaurant
Cloud: Use MongoDB Atlas connection string

ADMIN ACCESS
============
1. Register account
2. Update role to 'admin' in MongoDB
3. Access /admin dashboard

SEED DATA
=========
npm run seed (from backend folder)

NEXT STEPS
==========
1. Install dependencies
2. Configure .env
3. Start backend
4. Start frontend
5. Visit http://localhost:3000

Ready to serve! 🍽️
